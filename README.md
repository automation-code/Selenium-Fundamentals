# Selenium Fundamentals

Include all selenium classes and methods with examples:

```
1. Actions - ActionsUtility.java
2. Alerts - AlertsUtility.java
3. BrowserOptions - AppleSafari.java, GoogleChrome.java and MicrosoftEdge.java
4. Cookies - CookiesUtility.java
5. DevTools - CaptureDataSummery.java, FailedRequest.java, FinishedLoading.java, LogEntries.java, PerformanceMatrics.java,
              ReceivedResponse.java and SendRequest.java
6. Dropdown - DropdownUtility.java
7. Frame Handling - FrameUtility.java
8. JavaScript Executor - JavaScriptsAsync.java, JavaScriptsBasic.java, JavaScriptsBrowsers.java, JavaScriptsClicks.java,
                         JavaScriptsInformation.java, JavaScriptsManipulation.java, JavaScriptsNavigation.java,
                         JavaScriptsOperation.java, JavaScriptsScrolling.java and JavaScriptsUtility.java
9. Locators - LocatorsUtility.java
10. Screenshots - Screenshots.java, ScreenshotsBase64.java and ScreenshotsElement.java
11. Tables - DynamicTable.java, PaginationTable.java, StaticTable.java and TableUtility.java
12. Waits and Timeouts - PageLoadLoadStrategies.java, ThreadSleep.java and WaitsTimeouts.java
13. WebDrivers - Common.java
14. WebElements - WebElements.java
```

SeleniumLibrary - Contains all selenium custom methods for ready to use.
